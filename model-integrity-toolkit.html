How it fits

The Toolkit operates upstream of verification & validation, peer review, regulatory submission, and operational use. It does not replace downstream processes; it prevents structurally incoherent models from reaching them.

How to run it

A complete run requires only four artifacts:

  • Model purpose and scope
  • Inputs, parameters, and outputs (with units if declared)
  • Equations or logical relationships
